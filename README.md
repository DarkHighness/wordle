# Wordle

一个针对中文成语和诗词的 Wordle

## 特点:

* 同时支持成语和诗词的 Wordle

* 全离线使用, 不用担心任何记录信息

* 做不出来轻松 hack, 点放弃也能继续做

* 代码真的写的很**辣鸡**
